Popover

A high-quality, unstyled React popover component that displays an accessible popup anchored to a button.

Overview

The Popover component provides a flexible way to display contextual content that appears when triggered. It's perfect for tooltips, dropdowns, and contextual information that needs to be positioned relative to a trigger element.

✨ Key Features

  • • Accessible keyboard navigation
  • • Flexible positioning options
  • • Portal rendering for proper z-index stacking
  • • Focus management and trapping
  • • Customizable trigger and content
  • • Screen reader support with ARIA attributes

Basic Usage

Here's a simple example of how to use the Popover component:

Live Demo

Popover Title

This is the popover content. It can contain any HTML elements.

Code 
import * as React from 'react';
import { Popover } from '@base-ui-components/react/popover';

export default function ExamplePopover() {
  return (
    <Popover.Root>
      <Popover.Trigger className="trigger-button">
        Click me
      </Popover.Trigger>
      <Popover.Portal>
        <Popover.Positioner sideOffset={8}>
          <Popover.Popup className="popover-content">
            <Popover.Title>Popover Title</Popover.Title>
            <Popover.Description>
              This is the popover content. It can contain any HTML elements.
            </Popover.Description>
          </Popover.Popup>
        </Popover.Positioner>
      </Popover.Portal>
    </Popover.Root>
  );
}

API Reference

Popover.Root

The root component that provides the popover context and state management.

Props

Prop Type Default Description
open boolean false Controls whether the popover is open
onOpenChange (open: boolean) => void - Callback when open state changes

Popover.Trigger

The element that triggers the popover when clicked or activated.

Props

Prop Type Default Description
asChild boolean false Render as child element instead of button

Popover.Portal

Renders the popover content in a portal to ensure proper z-index stacking.

Props

Prop Type Default Description
container Element | null document.body Container element for the portal

Examples

Custom Positioning

Control the positioning of the popover with various options: 

<Popover.Positioner 
  side="top" 
  sideOffset={8}
  align="center"
  alignOffset={0}
>
  <Popover.Popup>
    Content positioned above trigger
  </Popover.Popup>
</Popover.Positioner>

Custom Content

Create rich content with custom styling and interactive elements: 

<Popover.Popup className="custom-popover">
  <Popover.Title>Settings</Popover.Title>
  <div className="space-y-2">
    <label className="flex items-center">
      <input type="checkbox" />
      <span className="ml-2">Enable notifications</span>
    </label>
    <button className="w-full bg-blue-500 text-white py-2 rounded">
      Save Changes
    </button>
  </div>
</Popover.Popup>

Accessibility

The Popover component is built with accessibility in mind and includes proper ARIA attributes, keyboard navigation, and screen reader support.

Accessibility Features

  • • ARIA attributes: Uses aria-expanded and aria-controls
  • • Keyboard navigation: Escape key closes the popover
  • • Focus management: Focus returns to trigger when closed
  • • Screen readers: Proper labeling and descriptions
  • • Portal rendering: Ensures proper z-index stacking
