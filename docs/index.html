ByDefault Documentation

This is the documentation for the ByDefault project. It contains guides, components, and utilities for building inclusive and accessible user interfaces.

Note: This documentation is inspired by Base UI's clean and accessible design patterns. Our focus is on neurodiversity, accessibility, and inclusive design.

Quick start

A quick guide to getting started with the ByDefault documentation and components.

Installation

npm i @base-ui-components/react

All components are included in a single package. Base UI is tree-shakeable, so your app bundle will contain only the components that you actually use.
